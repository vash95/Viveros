# Vivero - Web corporativa

Proyecto web corporativo moderno (React + Vite + Tailwind CSS) para presentar servicios, instalaciones y contacto de **Vivero Rabadán Ayuso**.

## Requisitos

- Node.js 18+
- npm 9+

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build de producción

```bash
npm run build
```

## Estructura principal

- `src/components/` componentes reutilizables de cada sección
- `src/data/services.js` datos editables de servicios
- `src/data/gallery.js` datos editables de galería

## Notas

- La galería y servicios están preparados para reemplazar imágenes placeholder por fotos reales.
- El mapa está integrado con iframe configurable.
- No incluye ecommerce, carrito, login ni pagos.
